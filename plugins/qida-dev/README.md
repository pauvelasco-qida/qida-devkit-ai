# qida-dev

Core development conventions and commands for Qida apps.

## Commands

### `/qida-dev:hello-qida`

Smoke test: confirms the plugin is installed and loaded, and lists the components it currently provides.

## Installation

From the qida-devkit-ai marketplace:

```
/plugin marketplace add pauvelasco-qida/qida-devkit-ai
/plugin install qida-dev@qida-devkit-ai
```
